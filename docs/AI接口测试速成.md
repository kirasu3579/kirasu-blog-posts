# AI接口测试速成

# 一、接口(API)是什么？

接口：全称是Application Programming Interface，中文翻译为**应用程序编程接口**。

作用：给前端与后端提供交互数据的**通道**。

步骤:

1、打开浏览器，按F12/fn+F12

2、切换到“Network”(网络)面板

3、勾选Preserve log（保留日志）
